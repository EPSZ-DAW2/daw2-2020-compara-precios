Proyecto en Grupo
Repositorio y Notas sobre el Control Versiones del Proyecto.

Repositorio en GITHUB para el trabajo en grupo:

URL → https://github.com/EPSZ-DAW2/daw2-2020-compara-precios

Se puede utilizar tanto un cliente de tipo SVN como uno de tipo GIT, aunque se recomienda el último por ser el que mejor se integra con el repositorio del proyecto, pero elegid el que queráis, es una elección particular de cada uno.

El usuario y la contraseña de acceso necesaria para escritura son propios de cada usuario y son los mismos datos que se utilizan para acceder a "github".
Para subir cambios al proyecto debéis estar como usuarios registrados del sistema "github" y haber sido agregados como colaboradores del proyecto.
Consultad con el profesor en caso de duda.
